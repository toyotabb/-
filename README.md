Разработан в качестве практического задания на курсе Профессия Fullstack веб-разработчик на JavaScript и PHP

* HTML

* CSS (Bootstrap)

## Как открыть сайт:

Зайти в папку сайта в файловом менеджере, кликнуть 2 раза по файлу index.html.
